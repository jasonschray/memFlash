User Profiles

User profiles are the root of this site. From this page all other parts of the site are accessible.

Profile Info

Profiles contain a profile photo, a description of the user, and then the date when the account was created. The profile photo and user profile description is editable through a button on the profile page. Please keep the description modest as it is publicly available to any user on the site. From this page, one can access their deckbook and favorited decks as well as update their profile.

Flashdecks

Flashdecks contain all flashcards in this site. To test yourself, you must use flashdecks.

Deck Book

A user's deck book is where all of their flashdecks are contained. Here a user can view their decks. A user can view other users deckbooks and see their decks as well. This book is publicly available. Here a user can also create a new deck.

Making A Deck

When a user decides to make a new flashdeck the user must first define the type of cards that will be contained within that deck. When a user clicks on the "Make New Deck" button, a pop-up window comes up. The first piece of data required is the name of the deck to be made. All cards in a deck must have the same number of sides, side types, and side names so the user must decide how many sides those cards will have along with their side names and data types.

Deleting A Deck

Be careful about deleting a deck because it is final once done.

Testing

There are two ways to use this site to learn things through self testing. Self testing is done either by shuffling through the cards or tracking your knowledge through the card tester feature. When you shuffle through the cards then you just click through the cards randomly and indefinitely. If you go through the tester option you must select the side to be questioned on, the side to be answering with and the type of answers to give (whether know/don't know or multiple choice). All testing is on the honor system as this is a tool one should use to teach themselves.

Flashcubes

Flashcubes are the name for the cards that go in flashdecks. They can have up to 4 sides.

Flashcube Capabilities

A flashcard is quite useful because it has 4 sides so you can store more than just two pieces of information about a fact. Also the sides of flashcubes are quite versatile because they can contain most language writing systems, images, and audio file formats.

Making A Flashcube

A flashcube is made by clicking the "Add A New Card" button on a flashdeck profile page. Then enter the data on the form that pops up.

Forums

We believe at this site that learning can be a very personal experience but we also believe that we all benefit from discussing thoughts and interacting with others. We aim to add a friends feature and messenger app in the future. But until then we have made a forumn where ideas can be discussed.

Creating A Forum

Creating a forum is something that anyone can do. Simply go to the Forums page and then click the "Create Forum" button. Then enter the forum title and description and press submit.

Posting On A Forum

To post on a forum, go to the last page of the forum and click the button "Create Post" to create a post.

Search

Search is a way for people to find other users as well as decks that contain certain topics and titles.

Search Options

Searches can be filtered by Users as well as Decks. In the future we plan to add forum posts.
